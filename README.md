## Setup Locally

### Frontend

- Navigate to `client` folder

```
cd client
```

- Install Dependencies

```
npm install
```

- Run `client app`

```
npm run dev
```

---

### Backend

- Navigate to server folder

```
cd server 
```

- Install Dependencies

```
npm install
```

- Run `server app`

```
npm run dev
```
